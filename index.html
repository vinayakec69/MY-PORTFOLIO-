SmartSeg: Plastic Waste Sorting

Abstract & Problem

The global increase in plastic waste necessitates automated and highly accurate sorting mechanisms for efficient recycling. Traditional facilities rely on manual labor or expensive optical sorters. SmartSeg is an Edge-AI based plastic waste sorting system designed for high-speed conveyor belt operations.

Hardware Architecture

  • NVIDIA Jetson Orin Nano: Provides accelerated AI inference, processing high-resolution frames locally to ensure minimal latency.
  • Intel RealSense Depth Camera: RGB stream is used for classification; infrared depth stream acts as a critical hardware gate to conserve GPU memory.

Proposed Methodology

  • Zero-Domain-Gap Dataset: Over 2,324 images were captured directly on the physical conveyor belt to prevent domain shift issues and improve real-world performance.
  • Depth Gating: The YOLO model is only triggered when the depth sensor detects an object physically present on the conveyor belt, saving immense computational power.
  • Majority Voting: Implemented a rolling buffer of the last 5 frames. The final classification decision is based on a majority vote, entirely eliminating single-frame hallucinations.

Experimental Results & Future Scope

By removing computationally expensive depth-RGB alignment operations and implementing the depth gate, the Jetson Orin Nano achieved a consistent processing speed of 31 Frames Per Second (FPS) with high accuracy across 6 material classes (PET, HDPE, PP, LDPE, MISC). Future iterations aim to integrate physical pneumatic actuators via an ESP32 microcontroller to physically sort the plastics.

KRUX: Gamified Smart Bin

Abstract & Problem

The inefficiency of current recycling pipelines stems from a lack of consumer motivation and severe cross-contamination. KRUX is a decentralized, gamified Smart Bin ecosystem designed to shift the sorting process to the consumer source by incentivizing users with 'KRUX Coins'.

View Live Project

Software & Hardware Architecture

  • Progressive Web App (PWA): Built with React 18, Vite, and Tailwind CSS. Provides a frictionless experience across iOS and Android.
  • ESP32 Microcontroller: Acts as the physical validator. Equipped with an IR obstacle sensor and a servo motor for routing plastic.
  • Firebase RTDB: Acts as a zero-latency communication bridge between the web app and the physical bin.

Proposed Methodology

  • On-Device Machine Learning: Uses TensorFlow.js (MobileNet) to perform image classification entirely on the edge (the user's mobile device). This eliminates cloud costs and guarantees total user privacy.
  • Secure Handshake Protocol:
    1. User scans QR to link session.
    2. In-browser ML classifies plastic and enters escrow.
    3. User drops plastic; ESP32 detects it and updates Firebase RTDB.
    4. PWA detects the state change and securely mints KRUX coins.

Experimental Results & Future Scope

The ecosystem demonstrates a scalable, cost-effective approach to decentralizing waste segregation, eliminating traditional backend bottlenecks while providing a highly engaging user experience. Future iterations aim to expand the on-device ML dataset for higher precision across multi-layered plastics.

CareLens: Spatial Navigation

Abstract & Problem

Traditional tools like the white cane provide physical feedback but lack semantic understanding (they can't tell the user *what* the object is). CareLens is an Edge-AI based spatial navigation and obstacle detection system designed for visually impaired individuals, providing real-time audio cues.

Hardware Architecture

  • NVIDIA Jetson Orin Nano: Local inference engine for processing high-resolution frames without cloud latency.
  • Intel RealSense Depth Camera: RGB for semantic material classification; Depth stream for calculating exact physical distances.
  • Zero-Latency Audio: Bypassed Bluetooth bottlenecks by routing audio via a Type-C DAC to In-Ear Monitors for instantaneous warnings.

Proposed Methodology

  • YOLOv8s Optimization: Pivoted to pre-trained YOLOv8s COCO weights to eliminate Out-Of-Memory (OOM) crashes in the CUDACachingAllocator, achieving flawless zero-shot detection of 80 classes.
  • Offline TTS Gating: Cloud APIs were replaced with pyttsx3 for 100% offline, zero-latency audio generation.
  • Intelligent Cooldowns: Implemented a 3-second rolling memory buffer so objects (e.g., "Chair ahead") do not trigger audio spam.
  • Spatial Navigation (NAV): Guides users to specific targets by evaluating bounding box X-coordinates against the frame thirds to issue "Turn Left" or "Turn Right" commands.

Experimental Results & Future Scope

By utilizing headless execution (--no-gui), the system achieved a consistent 30 FPS without triggering swap memory usage. The GTA-style navigation successfully guided users to targets within a 0.6-meter radius. Future iterations aim to deploy this architecture onto commercial Rokid AR smart glasses by quantizing the model to INT8 TensorFlow Lite.
